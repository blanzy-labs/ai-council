# AI Council v0.1.0 Scope

AI Council v0.1.0 is a local-first, text-based multi-persona council room for public testing, demos, and iteration.

## What v0.1.0 Does

- Runs a FastAPI backend and React/Vite frontend.
- Provides default council personas.
- Creates in-memory sessions.
- Runs controlled council responses with OpenAI or mock provider override.
- Supports follow-up chat with the whole council or one persona.
- Shows event-level live activity with Server-Sent Events.
- Exports the current in-memory transcript as Markdown or JSON.
- Clears transcript messages and recent events for a session.
- Starts and runs mock flows without `OPENAI_API_KEY`.

## Intentionally Not Included

- Voice
- Gemini
- Database persistence
- User accounts or login
- Telemetry
- Token-level provider streaming
- Long-term history
- Open-ended autonomous agent loops

## Roadmap

- v0.2.0: voice with OpenAI.
- v0.3.0: Gemini provider support.

## Known Limitations

- All session, transcript, event, and export state is in memory.
- Restarting the backend loses sessions and transcripts.
- Event streaming is event-level only, not token-by-token provider output.
- OpenAI calls require `OPENAI_API_KEY`; mock mode is the default testing path.
- The frontend has build-time validation but no dedicated frontend test runner yet.
