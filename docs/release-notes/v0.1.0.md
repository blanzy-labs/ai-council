# AI Council v0.1.0 - Local Council MVP

## Release Summary

AI Council v0.1.0 is the first local MVP: a text-based multi-persona council room for structured discussion, critique, follow-up chat, live activity events, and transcript export. It is designed for local demos and testing, with mock provider support so the full flow works without an OpenAI API key.

## What Is Included

- FastAPI backend
- React/Vite frontend
- Docker Compose local workflow
- Default persona registry
- In-memory session store
- In-memory transcript store
- In-memory recent event buffer
- OpenAI provider adapter
- Deterministic mock provider
- Controlled council run endpoint
- Follow-up chat endpoint
- Ask whole council flow
- Ask one persona flow
- Event-level Server-Sent Events stream
- Recent events endpoint
- Markdown export
- JSON export
- Transcript clearing
- Event clearing
- Health, root, and version metadata endpoints
- Backend tests and frontend build validation

## Intentionally Excluded

- Voice
- Gemini
- Database persistence
- User accounts or login
- Telemetry
- Hosted deployment
- Long-term session history
- Token-by-token model streaming
- Autonomous/open-ended agent loops

## Validation Checklist

- Backend tests pass with `cd backend && uv run pytest`.
- Frontend builds with `cd frontend && pnpm build`.
- Docker images build with `docker compose build`.
- Docker stack starts with `docker compose up --build`.
- Backend health returns ok.
- Frontend is reachable at http://localhost:5173.
- Mock session create/run/chat/export flow works.
- OpenAI missing-key response is clean.
- No real OpenAI calls run in normal tests.

## Known Limitations

- All sessions, messages, events, and exports are generated from runtime memory.
- Restarting the backend clears all in-memory state.
- SSE is event-level only, not token-level provider streaming.
- v0.1.0 is not hardened for public internet exposure.
- Frontend validation is currently build-based; no dedicated frontend test runner is included.

## Upgrade And Compatibility Notes

This is the first release. No upgrade path is required.
